# Netflix Movies & TV Shows — Exploratory Data Analysis

## Project Overview
Exploratory Data Analysis on the Netflix Movies & TV Shows dataset as part of the
Pluto Academy AI & ML Internship Program (Project 1).

## Dataset
- **Source:** Kaggle — Netflix Movies and TV Shows
- **Link:** https://www.kaggle.com/datasets/shivamb/netflix-shows
- **Size:** 8,807 titles × 12 columns

## Tools Used
Python · Pandas · NumPy · Matplotlib · Seaborn · Google Colab

## What This Project Covers
- Data loading, inspection, and cleaning
- Exploratory Data Analysis (5 key questions answered)
- 6+ visualizations (pie chart, bar chart, line chart, histogram, heatmap, scatter plot)
- 5 business insights derived from the analysis

## How to Run
Open the notebook in Google Colab and upload `netflix_titles.csv` from the dataset link above.
